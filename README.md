# Yahoo Bid Monitor

## Usage

- Sendiblue API key setup.
- Set the following CSV files:
  - booth.csv
  - mail_addresses.csv
  - mail_setting.csv
  - email_auth.txt

- Setting the Cron job to run the `bid_monitor.py` program to fetch bid data.
- Setting the Cron job to run the `email_client.py` program to send newsletter.
